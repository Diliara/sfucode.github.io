#cd-7 theme

Wrap your content in a div with a #cd-7 id

Banner

1 - With Text

<div class="banner">
<h2>#cd-3 theme</h2>
</div>

The .banner sets up default style for banner placeholder with grey background. The example with h2 is on the top of this page

2 - With Image

<div class="banner">
<img alt="banner" src=""/>
</div>

The banner image should be within .banner class. The rest of the images in this theme have automatic border.

Ordered List

Ordered list with extra space between list items (for listed items with a lot of content).

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3
  1. Item 1
    • Sub Item 1
    • Sub Item 2
  2. Item 2
  3. Item 3

Add .no-space class to ol or ul for no extra space.

  1. Item 1
  2. Item 2
  3. Item 3

Print Styles

Content wrapped in .hidden-print class will not be displayed for printed pages. Should be used for multimedia.

.page-break class splits content to the next page.

.no-split-for-print class can be used to avoid page break within different elements, such as tables.
