#cd-2 theme

Wrap your content in a div with a #cd-2 id

Banner

<div class="banner">
    <h2>Banner placeholder (like on the top of this page)</h2>
</div>

The .banner spans across the page width.

Typography

H4 Heading

H5 Heading

H2 is allocated for page title (usually within the banner). H3 are for main sections (section headings on this page)

H3s are automatically numbered.

<h3>Typography</h3>
<h4>H4 Heading</h4>
<h5>H5 Heading</h5>

Videos

    <iframe src="https://www.youtube.com/embed/XQu8TTBmGhA" width="560" height="315"></iframe>
</div>

.vid class provides space at the top & bottom of the video.

Print Styles

Content wrapped in .hidden-print class will not be displayed for printed pages. Should be used for multimedia.

.page-break class splits content to the next page.

.no-split-for-print class can be used to avoid page break within different elements, such as tables.
